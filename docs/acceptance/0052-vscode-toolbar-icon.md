# AT-0052: VS Code Toolbar Icon

## Purpose

Verify that the karasu toolbar icon appears in the VS Code editor title bar when a `.krs` file is open.

## Checklist

- [ ] Open a `.krs` file in VS Code
- [ ] The editor title toolbar shows the 鴉 kanji icon (not a text label)
- [ ] In a dark theme, the icon appears in light color (`#dce8ff`)
- [ ] In a light theme, the icon appears in dark color (`#1a1a2e`)
- [ ] Clicking the icon opens the karasu preview panel
