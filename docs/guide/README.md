# karasu ガイド

タスク指向の how-to ガイド集。構文の正確な仕様は [`docs/spec/`](../spec/)、設計思想は [`docs/concepts.md`](../concepts.md) を参照してください。各ガイドは英語版（`.md`）と日本語版（`.ja.md`）を持ちます。

| ガイド | 対象 | 向き | 内容 |
|--------|------|------|------|
| [サービス/チーム境界設計](service-team-design.ja.md)（[EN](service-team-design.md)） | アーキテクト | 設計（前向き） | ドメイン依存からのサービス分割、逆コンウェイ戦略、チーム別ファイル分割 |
| [オンボーディング](onboarding.ja.md)（[EN](onboarding.md)） | 中途入社・引き継ぎ | 読解（逆向き） | `translate` で既存資産から骨格を起こし、読み下しながら図にまとめる |

2 つのガイドは対になっています。境界設計ガイドは抽象から具体へ降ろす「これから設計する」道具としての使い方、オンボーディングガイドは具体から抽象へ上げる「既に在るものを読み解く」使い方を扱います。
