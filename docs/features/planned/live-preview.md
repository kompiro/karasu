# リアルタイムプレビュー

## 概要

`.krs` ファイルの編集内容をリアルタイムで SVG プレビューに反映する。

## UI構成

```
┌─────────────────┬─────────────────┐
│  Monaco Editor  │   SVGプレビュー  │
│  （左ペイン）   │  （右ペイン）   │
│                 │                 │
├─────────────────┴─────────────────┤
│  警告パネル                        │
└───────────────────────────────────┘
```

## 動作仕様

- エディタの変更を検知して即座にパースし直す
- パースエラーの場合はエラー箇所をエディタ上でハイライト
- 警告パネルにスタイル衝突・ドメイン分散などの警告を一覧表示

## 未決定事項

- デバウンス時間（入力後何msでパース実行するか）
- エラー時の挙動（直前の正常表示を維持するか、エラー表示にするか）
- 複数ファイルの同時編集
