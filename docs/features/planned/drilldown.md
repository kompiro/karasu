# ドリルダウン機能

## 概要

任意のノードをダブルクリックすると、そのノードの詳細図に遷移する。

## 動作の種類

### インラインネストの場合

ノードがインラインネストで記述されている場合、その場で展開してサブ図を表示する。

### 外部ファイル参照の場合

`import { X } from "file.krs"` で参照されているノードの場合、
該当ファイルを読み込んで遷移する。

## パンくずナビゲーション

ドリルダウンの階層を示すナビゲーションを表示する。

```
ECプラットフォーム > ECサイト > 受注
```
