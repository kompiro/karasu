# ADR-20260512-02: Dependabot Batch Triage (2026-05-12) — `pnpm/action-setup` 6.0.5

- **日付**: 2026-05-12（火曜）
- **ステータス**: 決定済み
- **関連**:
  - PR [#1345](https://github.com/kompiro/karasu/pull/1345) — `chore(deps): bump pnpm/action-setup from 6.0.3 to 6.0.5`（CI green、merge 済み）
  - [ADR-20260329-01](./20260329-01-dependabot.md) — Dependabot 採用
  - [ADR-20260414-01](./20260414-01-dependabot-batch-2026-04-14.md) — 前回のバッチ triage
  - [ADR-20260428-02](./20260428-02-update-dependencies-20260428.md) — 直近の依存更新バッチ
  - [ADR-20260512-01](./20260512-01-fast-uri-override-pin.md) — 同日の `fast-uri` security override（別件）

## 背景

2026-05-12 週の Dependabot version update バッチで起票された PR を triage した。今週起票されたのは 1 本のみ:

| PR | 対象 | エコシステム | 変更 | CI |
|----|------|--------------|------|----|
| [#1345](https://github.com/kompiro/karasu/pull/1345) | `pnpm/action-setup` 6.0.3 → 6.0.5 | github_actions | npm bootstrap 時の `PATH` 追加順序の修正（v6.0.4 / v6.0.5、prepend → append）| 全 check green |

npm 依存の version update PR は今週は起票されなかった（cooldown 7 日 — ADR-20260422-01 — の影響、および対象依存に新リリースがなかったため）。なお同日に `fast-uri` の transitive security pin を ADR-20260512-01 で別途処理しているが、それは Dependabot alert 起因の手動対応であり本バッチには含めない。

## 決定

- **PR #1345 を squash merge する**。`pnpm/action-setup` のパッチ更新で、CI ワークフローの pnpm セットアップ挙動の bugfix のみ。破壊的変更なし、全 required check green。
- `.github/dependabot.yml` は変更しない。

## 理由

- `pnpm/action-setup` は CI 専用の GitHub Action で、ランタイム成果物には影響しない。6.0.3 → 6.0.5 は同一 major・同一 minor のパッチ 2 件で、`PATH` 順序の修正のみ。
- 全 CI（Check / Reference docs / TPL validate / Validate / gitleaks）が green で、secret-gated job（Playwright / ExTester / Cloudflare）は bot PR ポリシー（ADR-20260428-01）通り skip。
- 単一・低リスクのパッチであり、バッチとして個別評価ではなくまとめて受け入れて差し支えない。

## 却下した案

特になし（マージ一択）。今週は判断を要する複数 PR の競合も、CI 失敗も、major 更新も発生しなかった。

## スコープ外

- 次回以降のバッチで複数 PR が競合する／major 更新が混ざる場合は、ADR-20260414-01・ADR-20260428-02 と同様にバッチ ADR で triage する。単一の自明なパッチであれば ADR を起こさず merge してよい（本 ADR は「今週分の対応記録を残したい」という要請に応じた記録）。
