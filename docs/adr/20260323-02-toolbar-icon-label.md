# ADR-20260323-02: ツールバーボタンはアイコン+テキストラベル必須

- **日付**: 2026-03-23
- **ステータス**: 決定済み

## 背景

ツールバーにアイコンのみのボタンを配置すると、ホバーやツールチップなしには機能が判断できない。

## 決定

ツールバーのすべてのボタンに **アイコン + テキストラベル** を組み合わせて表示する。

```
✓ 良い例:  ? Reference
✗ 悪い例:  ?
```

## 理由

- テキストラベルを併記することで、ユーザーがホバーやツールチップに頼らずに機能を把握できる
- アクセシビリティの向上

## 実装ルール

- ツールバーボタンには共通の `.toolbar-btn` クラスを使用する
- `.preview-toolbar` はプレビュー列の最上部に配置する
- ボタンは右寄せ（`justify-content: flex-end`）
- パンくずリストはツールバーの下に独立した行として配置する
