# ADR-20260312-02: ツール名「karasu」の採用

- **日付**: 2026-03-12
- **ステータス**: 決定済み

## 背景

テキストベースのアーキテクチャモデリングツールに名前をつけるにあたり、
既存ツール（Mermaid、PlantUML など）と差別化できる独自の名前を検討した。

## 検討した候補

- `huginn`：北欧神話の鴉（思考）→ 自動化ツールとして既に存在する
- `muninn`：北欧神話の鴉（記憶）→ 監視ツールとして既に存在する
- `karasu`：日本語で「鴉」

## 決定

**karasu**（鴉）を採用する。

## 理由

- 既存ツールとの競合がない
- 北欧神話のオーディンの使い魔ヒギン・ムニン（思考と記憶の鴉）に由来する
- 世界を俯瞰して情報を集め、必要な場所へ降りていく鴉の姿が、
  ドリルダウン型アーキテクチャ把握のコンセプトと重なる
- 作者の自転車「ヒギン」にも由来する

## ファイル拡張子

- `.krs`：図の定義ファイル
- `.krs.style`：スタイル定義ファイル
