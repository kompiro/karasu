# ツールバー UI デザインルール

- **日付**: 2026-03-23
- **ステータス**: 採用

## ルール

### ボタンには必ずアイコン+テキストラベルを付ける

ツールバーのボタンは「アイコンだけ」ではなく、**アイコン + テキストラベル** の形式で表示する。

```
✓ 良い例:  ? Reference
✗ 悪い例:  ?
```

**理由**: アイコンのみのボタンは機能が不明瞭。テキストラベルを併記することで、ユーザーがホバーやツールチップに頼らずに機能を把握できる。

### CSS クラス

ツールバーボタンには共通の `.toolbar-btn` クラスを使用する。

```css
.toolbar-btn {
  background: #334155;
  color: #94a3b8;
  border: 1px solid #475569;
  border-radius: 4px;
  padding: 2px 8px;
  font-size: 11px;
}
```

### ツールバーの配置

- `.preview-toolbar` はプレビュー列の最上部に配置する
- ボタンは右寄せ（`justify-content: flex-end`）
- パンくずリストはツールバーの下に独立した行として配置する
